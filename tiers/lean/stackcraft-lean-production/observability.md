# Observability

## Minimum requirements

- Centralized application logs
- Infrastructure metrics
- Uptime monitoring
- Alerting on critical failures

## What to track

- CPU, memory, disk
- App error rate
- HTTP 5xx responses
- Restart frequency
- DB availability
- Latency trends

## Alerts

- Production app unavailable
- High error rate
- Disk nearing full
- Database connection failures
- Failed backup jobs
