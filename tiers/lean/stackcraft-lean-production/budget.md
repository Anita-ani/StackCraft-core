# Lean Production Budget

## Typical range

$400–$700 per month (USD, indicative).

## Budget areas

- Compute
- Managed database
- Storage / backups
- Monitoring / logging
- Alerting
- DNS / CDN / WAF
- CI/CD

## Example allocation (illustrative)

- App compute: $80–$180
- Managed DB: $100–$250
- Monitoring / logging: $20–$100
- CDN / DNS / security: $20–$80
- Backups / storage: $20–$60
- Staging: $80–$150

## Notes

Lean Production is where cost starts buying **operational safety**, not just capacity.
