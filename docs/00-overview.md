# Overview

StackCraft Core is a **cloud-neutral** reference for production architecture and operations. It ties **budget tiers**, **team size**, and **risk** to concrete expectations for availability, recovery, and observability.

For directory layout, see [09-repo-structure.md](09-repo-structure.md). For the high-level entry point, see [README.md](../README.md).

## Ecosystem

| Repository        | Role |
|-------------------|------|
| **stackcraft-core** (this repo) | Tiers, frameworks, checklists, upgrade paths |
| **stackcraft-sectors** | Industry-specific baselines and constraints |
| **stackcraft-setups** | Environment- and tool-specific implementation |
