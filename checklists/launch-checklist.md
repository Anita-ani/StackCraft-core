# Production Launch Checklist

## Infrastructure

- [ ] Compute provisioned  
- [ ] Database configured  
- [ ] DNS configured  
- [ ] SSL enabled  

## Security

- [ ] Secrets not in code  
- [ ] Firewall configured  
- [ ] Access control defined  

## Observability

- [ ] Logs collected  
- [ ] Metrics tracked  
- [ ] Alerts configured  

## Backups

- [ ] Backup configured  
- [ ] Restore tested  

## Deployment

- [ ] CI/CD pipeline working  
- [ ] Rollback strategy defined
